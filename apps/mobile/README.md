# MyFreeMusic - Mobile App

Ứng dụng di động cho MyFreeMusic (Android & iOS).

## Công nghệ
- React Native

## Cài đặt
```bash
# Từ thư mục MyFreeMusic-FE
npm install
npm run dev:mobile
```

## Build
- Android: `npx react-native run-android`
- iOS: `npx react-native run-ios`
