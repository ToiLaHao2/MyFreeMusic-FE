# MyFreeMusic - Web App

Giao diện web cho MyFreeMusic.

## Công nghệ
- React / Next.js
- HLS.js (phát nhạc streaming)

## Cài đặt
```bash
# Từ thư mục MyFreeMusic-FE
npm install
npm run dev:web
```
